# Variable Types

## Variable Assignment

```sh
node
```

```js
// string
h = "hello"
w = " world!"
h + w // "hello world!"

// number
my_num = 8
my_num * 7 // 56

// boolean
my_bool = true
my_bool * 8 // 8
my_bool = false
my_bool * 7 // 0
```

## Basic Types

- String
- Number
- Boolean

```js
typeof(true) // 'boolean'
typeof(1) // 'number'
typeof("1") // 'string'
```

## Calculator

```js

```

## Boolean Ascertained

